# 从canvas2D 开始

## 创建和获取canvas

**html**
`<canvas></canvas>`

```js
const canvas = document.querySelector("canvas");
const ctx = canvas.getContext("2d");
```

## 添加图片

```js
const img = new Image();
img.src = "https://example.com/image.jpg";

img.onload = () => {
  ctx.drawImage(img, 0, 0);
};
```

## 添加文字

```js
ctx.font = "30px Arial";
ctx.fillStyle = "red";
ctx.fillText("Hello, World!", 10, 50);
```
