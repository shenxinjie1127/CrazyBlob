дәӢд»¶еҫӘзҺҜжңәеҲ¶

д»Җд№ҲжҳҜдәӢд»¶еҫӘзҺҜ

JavaScript жҳҜдёҖй—ЁеҚ•зәҝзЁӢгҖҒйқһйҳ»еЎһзҡ„иҜӯиЁҖгҖӮдәӢд»¶еҫӘзҺҜпјҲEvent LoopпјүжҳҜ JavaScript е®һзҺ°ејӮжӯҘзј–зЁӢзҡ„ж ёеҝғжңәеҲ¶пјҢе®ғиҙҹиҙЈеҚҸи°ғдё»зәҝзЁӢжү§иЎҢеҗҢжӯҘд»Јз Ғе’Ңз®ЎзҗҶејӮжӯҘд»»еҠЎзҡ„жү§иЎҢйЎәеәҸгҖӮ

дёәд»Җд№ҲйңҖиҰҒдәӢд»¶еҫӘзҺҜ

JavaScript еҸӘжңүдёҖдёӘдё»зәҝзЁӢпјҢеҗҢжӯҘд»Јз Ғдјҡйҳ»еЎһдё»зәҝзЁӢгҖӮдәӢд»¶еҫӘзҺҜзҡ„дҪңз”ЁжҳҜеңЁдё»зәҝзЁӢз©әй—Іж—¶пјҢд»Һд»»еҠЎйҳҹеҲ—дёӯеҸ–еҮәд»»еҠЎжү§иЎҢпјҢд»ҺиҖҢе®һзҺ°гҖҢйқһйҳ»еЎһгҖҚзҡ„зү№жҖ§гҖӮ

жү§иЎҢж Ҳе’Ңд»»еҠЎйҳҹеҲ—

жү§иЎҢж ҲпјҲCall Stackпјү

жү§иЎҢж ҲжҳҜдёҖдёӘеҗҺиҝӣе…ҲеҮәпјҲLIFOпјүзҡ„ж•°жҚ®з»“жһ„пјҢз”ЁдәҺеӯҳеӮЁд»Јз Ғжү§иЎҢж—¶зҡ„дёҠдёӢж–ҮгҖӮ

function fn1() {
  console.log("fn1 ејҖе§Ӣ");
  fn2();
  console.log("fn1 з»“жқҹ");
}

function fn2() {
  console.log("fn2 жү§иЎҢ");
}

fn1();
console.log("е…ЁеұҖжү§иЎҢе®ҢжҲҗ");

жү§иЎҢиҝҮзЁӢпјҡ

  1. е…ЁеұҖд»Јз Ғиҝӣе…Ҙжү§иЎҢж Ҳ
  2. fn1 иҝӣе…Ҙжү§иЎҢж Ҳ вҶ’ иҫ“еҮә "fn1 ејҖе§Ӣ"
  3. fn2 иҝӣе…Ҙжү§иЎҢж Ҳ вҶ’ иҫ“еҮә "fn2 жү§иЎҢ"
  4. fn2 жү§иЎҢе®ҢжҜ•еҮәж Ҳ
  5. fn1 з»§з»ӯжү§иЎҢ вҶ’ иҫ“еҮә "fn1 з»“жқҹ"
  6. fn1 еҮәж Ҳ вҶ’ иҫ“еҮә "е…ЁеұҖжү§иЎҢе®ҢжҲҗ"

д»»еҠЎйҳҹеҲ—пјҲTask Queueпјү

д»»еҠЎйҳҹеҲ—жҳҜдёҖдёӘе…Ҳиҝӣе…ҲеҮәпјҲFIFOпјүзҡ„ж•°жҚ®з»“жһ„пјҢз”ЁдәҺеӯҳеӮЁеҫ…жү§иЎҢзҡ„ејӮжӯҘеӣһи°ғгҖӮJavaScript дёӯжңүеӨҡз§Қд»»еҠЎйҳҹеҲ—пјҡ

  • е®Ҹд»»еҠЎйҳҹеҲ—пјҲMacro Task QueueпјүпјҡsetTimeoutгҖҒsetIntervalгҖҒI/OгҖҒUI Rendering
  • еҫ®д»»еҠЎйҳҹеҲ—пјҲMicro Task QueueпјүпјҡPromiseгҖҒMutationObserverгҖҒqueueMicrotask

з»Ҹе…ёе®һдҫӢи®Іи§Ј

е®һдҫӢ 1пјҡеҹәзЎҖдәӢд»¶еҫӘзҺҜ

console.log("1"); // еҗҢжӯҘд»Јз Ғ

setTimeout(() => {
  console.log("2");
}, 0); // е®Ҹд»»еҠЎ

Promise.resolve().then(() => {
  console.log("3");
}); // еҫ®д»»еҠЎ

console.log("4"); // еҗҢжӯҘд»Јз Ғ

жү§иЎҢйЎәеәҸпјҡ

иҫ“еҮә: 1 вҶ’ 4 вҶ’ 3 вҶ’ 2

еҲҶжһҗпјҡ

  1. йҰ–е…Ҳжү§иЎҢеҗҢжӯҘд»Јз Ғпјҡиҫ“еҮә 1пјҢиҫ“еҮә 4
  2. еҫ®д»»еҠЎ Promise.then еҠ е…Ҙеҫ®д»»еҠЎйҳҹеҲ—
  3. setTimeout еҠ е…Ҙе®Ҹд»»еҠЎйҳҹеҲ—
  4. еҗҢжӯҘд»Јз Ғжү§иЎҢе®ҢжҜ•пјҢжЈҖжҹҘеҫ®д»»еҠЎйҳҹеҲ—
  5. жү§иЎҢеҫ®д»»еҠЎпјҡиҫ“еҮә 3
  6. еҫ®д»»еҠЎйҳҹеҲ—жё…з©әпјҢжү§иЎҢе®Ҹд»»еҠЎпјҡиҫ“еҮә 2

е®һдҫӢ 2пјҡеөҢеҘ—зҡ„ Promise

console.log("1");

Promise.resolve()
  .then(() => {
    console.log("2");
    Promise.resolve().then(() => {
      console.log("3");
    });
  })
  .then(() => {
    console.log("4");
  });

console.log("5");

жү§иЎҢйЎәеәҸпјҡ

иҫ“еҮә: 1 вҶ’ 5 вҶ’ 2 вҶ’ 3 вҶ’ 4

еҲҶжһҗпјҡ

  1. иҫ“еҮә 1
  2. Promise.resolve() з«ӢеҚі resolvedпјҢз¬¬дёҖдёӘ .then еҠ е…Ҙеҫ®д»»еҠЎйҳҹеҲ—
  3. иҫ“еҮә 5пјҲеҗҢжӯҘд»Јз Ғпјү
  4. жү§иЎҢз¬¬дёҖдёӘеҫ®д»»еҠЎпјҡиҫ“еҮә 2
  5. еөҢеҘ—зҡ„ Promise.resolve() зҡ„ .then еҠ е…Ҙеҫ®д»»еҠЎйҳҹеҲ—
  6. жү§иЎҢз¬¬дәҢдёӘеҫ®д»»еҠЎпјҡиҫ“еҮә 3
  7. з¬¬дёҖдёӘ .then зҡ„еҗҺз»ӯ .then еҠ е…Ҙеҫ®д»»еҠЎйҳҹеҲ—
  8. жү§иЎҢз¬¬дёүдёӘеҫ®д»»еҠЎпјҡиҫ“еҮә 4

е®һдҫӢ 3пјҡasync/await

console.log("1");

async function asyncFn() {
  console.log("2");
  await console.log("3");
  console.log("4");
}

asyncFn();

console.log("5");

жү§иЎҢйЎәеәҸпјҡ

иҫ“еҮә: 1 вҶ’ 2 вҶ’ 3 вҶ’ 5 вҶ’ 4

еҲҶжһҗпјҡ

Node.js зү№жңү API

setImmediate

  console.log("setImmediate");
});

setTimeout(() => {
  console.log("setTimeout");
}, 0);

еңЁ Node.js дёӯпјҢsetTimeout е’Ң setImmediate зҡ„жү§иЎҢйЎәеәҸдёҚзЎ®е®ҡпјҢеҸ–еҶідәҺзі»з»ҹжҖ§иғҪгҖӮ

process.nextTick

process.nextTick(() => {
  console.log("nextTick");
});

Promise.resolve().then(() => {
  console.log("Promise");
});

жү§иЎҢйЎәеәҸпјҡ

иҫ“еҮә: nextTick вҶ’ Promise

process.nextTick зҡ„дјҳе…Ҳзә§й«ҳдәҺ PromiseгҖӮ

жҖ»з»“

дәӢд»¶еҫӘзҺҜж ёеҝғиҰҒзӮ№

  1. еҗҢжӯҘд»Јз Ғдјҳе…Ҳжү§иЎҢпјҡдё»зәҝзЁӢдјҡе…Ҳжү§иЎҢе®ҢжүҖжңүеҗҢжӯҘд»Јз Ғ
  2. жё…з©әеҫ®д»»еҠЎйҳҹеҲ—пјҡжҜҸж¬ЎдәӢд»¶еҫӘзҺҜејҖе§Ӣж—¶пјҢдјҡжү§иЎҢе®ҢжүҖжңүеҫ®д»»еҠЎ
  3. дёҖдёӘе®Ҹд»»еҠЎжү§иЎҢеҗҺжЈҖжҹҘеҫ®д»»еҠЎпјҡе®Ҹд»»еҠЎжү§иЎҢе®ҢеҗҺпјҢз«ӢеҚіжЈҖжҹҘе№¶жү§иЎҢжүҖжңүеҫ®д»»еҠЎ
  4. дёҚж–ӯеҫӘзҺҜпјҡйҮҚеӨҚгҖҢжё…з©әеҫ®д»»еҠЎ вҶ’ жү§иЎҢдёҖдёӘе®Ҹд»»еҠЎгҖҚзҡ„иҝҮзЁӢ
  5. UI жёІжҹ“ж—¶жңәпјҡеңЁеҫ®д»»еҠЎйҳҹеҲ—жё…з©әеҗҺгҖҒе®Ҹд»»еҠЎжү§иЎҢеүҚеҸҜиғҪи§ҰеҸ‘жёІжҹ“

и®°еҝҶеҸЈиҜҖ

е…ҲеҗҢжӯҘпјҢеҗҺеҫ®д»»еҠЎпјҢжңҖеҗҺе®Ҹд»»еҠЎпјҢдёҖдёӘдёҖдёӘжқҘ

еёёи§ҒиҖғзӮ№

  1. Promise vs setTimeout зҡ„жү§иЎҢйЎәеәҸ
  2. async/await зҡ„жң¬иҙЁжҳҜ Promise + дәӢд»¶еҫӘзҺҜ
  3. async еҮҪж•°дёӯзҡ„ await дјҡйҳ»еЎһеҗҺз»ӯд»Јз Ғжү§иЎҢ
  4. Node.js зҡ„ process.nextTick дјҳе…Ҳзә§й«ҳдәҺ Promise
  5. дәӢд»¶еҫӘзҺҜжҳҜ JavaScript е®һзҺ°ејӮжӯҘзҡ„еҹәзЎҖ
